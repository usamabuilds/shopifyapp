# Phase 1 First-Run Onboarding & Help (Merchant-Facing)

> Use this as the first-run guide for beta merchants using the current UI.

## Before you begin

This app provides WhatsApp messaging foundations for Shopify operations. In this phase, you are configuring and operating core flows (not advanced CRM/chatbot functionality).

## Step 1: Open Dashboard

- Go to **Dashboard** to see setup progress and readiness indicators.
- Use quick links to jump to settings, automations, templates, campaigns, and workflows.

## Step 2: Complete Settings

- Open **Settings**.
- Save merchant profile defaults (display name, support email/phone, timezone).
- Review checklist/readiness states and warnings.

## Step 3: Confirm Templates

- Open **Templates**.
- Review mapping status for each flow.
- Ensure template keys and categories align:
  - Utility: order confirmation, order status updates
  - Marketing: cart recovery, broadcast campaigns

## Step 4: Configure Automations

- Open **Automations**.
- Configure and save:
  - Order confirmation
  - Order status updates (all status template keys)
  - Abandoned cart recovery
- If an automation is blocked, resolve missing template keys first.

## Step 5: Set up Campaigns (optional at first-run)

- Open **Campaigns**.
- Create a draft campaign first.
- For scheduling, choose a future datetime and confirm template coverage.

## Step 6: Validate Workflow Foundation (optional)

- Open **Workflows**.
- Create or edit a workflow draft.
- Run preview execution to confirm validation before publishing.

## Step 7: Operational Check

- Open **Support tools**.
- Confirm webhook and outbound sections load.
- Use support actions only when recovering existing due work.

## Step 8: Monitor Outcomes

- Open **Analytics** for outcome summaries by messaging/campaign activity.

## Merchant safety reminder (Phase 1)

Merchants/operators are responsible for ensuring message eligibility and consent scope for utility vs marketing sends.
